# Secure Chat

Modern real-time secure chat application built using:

- Firebase Realtime Database
- Firebase Authentication
- WebRTC
- Cloudinary
- PWA Support

## Features

- Real-time messaging
- Voice & Video calls
- Status system
- Friend requests
- Seen ticks
- Typing indicator
- Media sharing
- Screenshot alerts
- Notifications
- Mobile responsive UI
